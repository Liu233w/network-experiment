待办事项管理系统
刘书敏 2015303087 14011501

本项目使用 JDK 1.8 编译，IDE 为 Intellij IDEA

服务端：
    保存在 server 目录中。如果您没有 Intellij IDEA，可以双击 server 目录中的 run.bat 来启动程序。

客户端：
    保存在 client 目录中。如果您没有 Intellij IDEA，可以双击 client 目录中的 run.bat 来启动程序。在启动程序后有相应的说明。

服务端会占用 8888 端口。请确保此端口正常。

wsdl路径：
http://localhost:8888/TodoList?wsdl

您也可以直接访问 wsdl 目录来查看wsdl文件

commands.txt 是我用来保存生成 wsdl 相关文件的命令。
